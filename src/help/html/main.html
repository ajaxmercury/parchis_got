Identificación del software y problema a resolver:

Parchís es una aplicación propuesta para la informatización del juego del parchís original, mediante la didáctica proporcionada por los recursos informáticos, logrando un mejor confort de los usuarios jugadores respecto al juego.

 

Elementos componentes:

La aplicación cuenta con una ventana principal, o tablero de juego, en el cual se juega con los dados y las fichas de cada jugador. Posee una ayuda indexada para la consulta de los usuarios principiantes en el juego. Además, permite guardar y abrir una partida anteriormente iniciada.

 

Organización y estructura:

Este manual de usuario o ayuda, cuenta con información referente a datos del desarrollo del juego informático. Materiales de consulta para usuarios principiantes, así como las principales reglas del juego y las instrucciones del mismo.

 

Forma de usar el manual:

Este manual puede ser usado para consultar los datos referentes al desarrollo de la aplicación, obtener información de como contactar con los desarrolladores y además, consultar los manuales de intrucciones y reglas del juego.
